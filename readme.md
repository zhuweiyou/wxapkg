# wxapkg

PC 微信小程序一键解密和解包

## 使用方法

1. 在 [release](https://github.com/zhuweiyou/wxapkg/releases/) 下载 `wxapkg.exe` 文件

2. PC 微信 -> 设置 -> 文件管理 -> 打开文件夹 -> 找到小程序文件 `/WeChat Files/Applet/{wxid}/{n}/__APP__.wxapkg`

> 微信 4.0 之后, 小程序文件的目录在 C:\Users\用户名\AppData\Roaming\Tencent\xwechat\radium\Applet\packages\

3. 鼠标拖动 `__APP__.wxapkg` 文件到 `wxapkg.exe` 即可完成解包, 将在当前目录输出解包后的文件夹
